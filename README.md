\# 💳 ATM Management System



A simple \*\*ATM Management System\*\* built using Java and MySQL that simulates basic banking operations like login, balance check, deposit, and withdrawal.



\---



\## 🚀 Features



\* 🔐 User Authentication (Login System)

\* 💰 Balance Inquiry

\* ➕ Deposit Money

\* ➖ Withdraw Money

\* 🧾 Transaction Handling

\* 🗄️ Database Integration (MySQL)



\---



\## 🛠️ Technologies Used



\* Java (Swing / AWT for UI)

\* MySQL (Database)

\* JDBC (Database Connectivity)



\---

\## ⚙️ How to Run



1\. Clone the repository:



```

git clone https://github.com/your-username/ATM-Management.git

```



2\. Open the project in your IDE (Eclipse / IntelliJ)



3\. Add MySQL Connector (JDBC Driver)



4\. Setup database:



&#x20;  \* Create database in MySQL

&#x20;  \* Update credentials in `DBConnection.java`



5\. Run `Main.java`
